# Plan: Outreach Intelligence + Message Drafts

**Date:** 2026-03-11
**Features:** C (Outreach Intelligence) + E (Cover Letter / Outreach Drafts)

## Goal

For each top match in the brief, produce:
1. **Outreach intel** — who to contact, how to find them, mutual connections angle
2. **Outreach draft** — a short, personalized LinkedIn message or email to the hiring manager

Both appear in the brief and are saved alongside the tailored resume.

## Design Decisions

### What goes where

| Artifact | Location | Generated by |
|----------|----------|-------------|
| Outreach intel + draft | `resumes/drafts/YYYY-MM-DD-company-track.outreach.md` | New `processors/outreach.py` |
| Outreach path | `jobs.outreach_path` column | Pipeline writes after generation |
| Brief section | Inline under each top match | `brief.py` reads path from job dict |

### DB schema change

Add `outreach_path TEXT` to `jobs` table — same pattern as `tailored_resume_path`. Provides:
- Idempotency: skip jobs that already have outreach (across days/runs)
- Simple brief rendering: `job["outreach_path"]` instead of file globbing

### What the LLM produces

One call per job (bundled prompt), returning:
```json
{
  "hiring_signals": {
    "hiring_manager_likely_title": "VP Engineering / CTO",
    "team": "Engineering",
    "how_to_find": "LinkedIn search: site:linkedin.com/in 'Posit' 'VP Engineering'",
    "linkedin_search_url": "https://www.linkedin.com/search/results/people/?keywords=..."
  },
  "outreach_draft": {
    "subject": "Re: VP Engineering at Posit",
    "message": "Hi [Name], ...",
    "channel": "linkedin_message",
    "personalization_hooks": ["open source background", "AI transformation focus"]
  }
}
```

### Scope control

- **No LinkedIn scraping.** We generate a search URL so the user can look up the right person manually. Scraping LinkedIn contacts is both fragile and risky.
- **No email lookup.** No Hunter.io / Apollo integration. Keep it simple.
- **One LLM call per job.** The prompt includes job description + company enrichment (already available). No new external API calls.
- **Skip recruiter listings.** `is_job_board()` companies get no outreach — there's no real hiring manager to find.

```python
"""Outreach intelligence and draft generation.

For each top-scored job, generates:
1. Hiring signals — who to contact, how to find them
2. Outreach draft — personalized LinkedIn message or email
"""
import json
import logging
import re
from concurrent.futures import ThreadPoolExecutor, as_completed
from dataclasses import dataclass, field
from pathlib import Path

from shortlist import llm

logger = logging.getLogger(__name__)


@dataclass
class OutreachResult:
    hiring_manager_title: str
    team: str
    how_to_find: str
    linkedin_search_url: str
    subject: str
    message: str
    channel: str  # linkedin_message, email, or linkedin_inmail
    personalization_hooks: list[str] = field(default_factory=list)


OUTREACH_PROMPT = """You are helping a senior engineering leader craft outreach for a job application.

## Candidate Context
{fit_context}

## Job
**Title:** {title}
**Company:** {company}
**Description:**
{description}

## Company Intel
{enrichment}

## Instructions

Generate two things:

### 1. Hiring Signals
Figure out who the hiring manager likely is:
- What title would they have? (e.g., "CTO", "SVP Engineering", "VP Platform")
- What team/org is this under?
- How to find them: suggest a specific LinkedIn search query
- Generate the LinkedIn search URL (use https://www.linkedin.com/search/results/people/?keywords=<company>%20<likely title>)

### 2. Outreach Draft
Write a SHORT outreach message (3-5 sentences max) for LinkedIn or email:
- Open with something specific about the company or role (not generic flattery)
- Connect one concrete thing from the candidate's background to what the role needs
- Close with a low-pressure ask (quick call, coffee chat)
- Tone: confident but not arrogant, specific not generic, peer-to-peer not supplicant

Return ONLY a JSON object:
```json
{{
    "hiring_signals": {{
        "hiring_manager_likely_title": "<title>",
        "team": "<team/org name>",
        "how_to_find": "<plain language search instructions>",
        "linkedin_search_url": "<full URL>"
    }},
    "outreach_draft": {{
        "subject": "<email subject or linkedin connection note>",
        "message": "<the 3-5 sentence message>",
        "channel": "linkedin_message",
        "personalization_hooks": ["<hook 1>", "<hook 2>"]
    }}
}}
```"""


def generate_outreach(
    title: str,
    company: str,
    description: str,
    enrichment_json: str | None,
    fit_context: str,
) -> OutreachResult | None:
    """Generate outreach intel + draft for a job."""
    enrichment_str = enrichment_json or "No company intel available."

    prompt = OUTREACH_PROMPT.format(
        fit_context=fit_context,
        title=title,
        company=company,
        description=description[:3000],
        enrichment=enrichment_str,
    )

    result = llm.call_llm(prompt)
    if not result:
        return None

    try:
        data = llm.parse_json(result)
    except (json.JSONDecodeError, ValueError):
        logger.error(f"Failed to parse outreach response for {company}/{title}")
        return None

    try:
        signals = data.get("hiring_signals", {})
        draft = data.get("outreach_draft", {})
        return OutreachResult(
            hiring_manager_title=signals.get("hiring_manager_likely_title", ""),
            team=signals.get("team", ""),
            how_to_find=signals.get("how_to_find", ""),
            linkedin_search_url=signals.get("linkedin_search_url", ""),
            subject=draft.get("subject", ""),
            message=draft.get("message", ""),
            channel=draft.get("channel", "linkedin_message"),
            personalization_hooks=draft.get("personalization_hooks", []),
        )
    except Exception as e:
        logger.error(f"Failed to build OutreachResult for {company}/{title}: {e}")
        return None


def save_outreach(
    result: OutreachResult,
    output_dir: Path,
    company: str,
    track: str,
    date: str,
) -> Path:
    """Save outreach intel + draft to a markdown file."""
    output_dir.mkdir(parents=True, exist_ok=True)

    safe_company = re.sub(r"[^a-z0-9]+", "-", company.lower()).strip("-")
    filename = f"{date}-{safe_company}-{track}.outreach.md"
    path = output_dir / filename

    lines = [
        f"# Outreach — {company}",
        "",
        "## Who to Contact",
        "",
        f"**Likely hiring manager:** {result.hiring_manager_title}",
        f"**Team:** {result.team}",
        f"**How to find:** {result.how_to_find}",
        f"**LinkedIn search:** [Find them]({result.linkedin_search_url})",
        "",
        "## Draft Message",
        "",
        f"**Channel:** {result.channel}",
        f"**Subject:** {result.subject}",
        "",
        result.message,
        "",
        "## Personalization Hooks",
        "",
    ]
    for hook in result.personalization_hooks:
        lines.append(f"- {hook}")
    lines.append("")

    path.write_text("\n".join(lines))
    logger.info(f"Saved outreach: {path}")
    return path


def outreach_jobs_parallel(
    jobs: list[tuple[int, str, str, str, str, str | None, str]],
    drafts_dir: Path,
    today: str,
    max_workers: int = 10,
) -> list[tuple[int, OutreachResult | None, Path | None]]:
    """Generate outreach for multiple jobs in parallel.

    jobs: list of (job_id, title, company, description, matched_track, enrichment_json, fit_context)
    Returns list of (job_id, result, path).
    """
    results = []

    def _process(job_id, title, company, description, track, enrichment_json, fit_context):
        result = generate_outreach(title, company, description, enrichment_json, fit_context)
        if result:
            path = save_outreach(result, drafts_dir, company, track, today)
            return (job_id, result, path)
        return (job_id, None, None)

    with ThreadPoolExecutor(max_workers=max_workers) as pool:
        futures = {
            pool.submit(_process, *job): job[0]
            for job in jobs
        }
        for future in as_completed(futures):
            try:
                results.append(future.result())
            except Exception as e:
                job_id = futures[future]
                logger.warning(f"Outreach generation failed for job {job_id}: {e}")
                results.append((job_id, None, None))
    return results
```

@cli.command()
@click.argument("company", required=False)
@click.option("--config", "config_path", default=None, help="Path to profile.yaml")
def outreach(company, config_path):
    """Generate outreach drafts for top matches (or a specific company)."""
    from shortlist.config import validate_config, validate_env
    from shortlist.processors.outreach import outreach_jobs_parallel
    from shortlist.processors.enricher import is_job_board
    from shortlist import llm
    from datetime import datetime

    config, root = _get_config(config_path)
    errors = validate_config(config, root) + validate_env(root, config)
    if errors:
        click.echo("❌ Fix config issues first.", err=True)
        raise SystemExit(1)

    llm.configure(config.llm.model)

    db = get_db(root / "jobs.db")
    drafts_dir = root / "resumes" / "drafts"
    today = datetime.now().strftime("%Y-%m-%d")

    if company:
        rows = db.execute(
            "SELECT id, title, company, description, matched_track, enrichment FROM jobs "
            "WHERE company LIKE ? AND status = 'scored' ORDER BY fit_score DESC",
            (f"%{company}%",),
        ).fetchall()
    else:
        rows = db.execute(
            "SELECT id, title, company, description, matched_track, enrichment FROM jobs "
            "WHERE status = 'scored' AND outreach_path IS NULL "
            "AND enrichment IS NOT NULL "
            "ORDER BY fit_score DESC LIMIT ?",
            (config.brief.top_n,),
        ).fetchall()

    inputs = [
        (r["id"], r["title"], r["company"], r["description"] or "",
         r["matched_track"] or "em", r["enrichment"], config.fit_context)
        for r in rows
        if not is_job_board(r["company"])
    ]

    if not inputs:
        click.echo("No jobs need outreach.")
        return

    click.echo(f"Generating outreach for {len(inputs)} jobs...")
    results = outreach_jobs_parallel(inputs, drafts_dir, today)

    count = 0
    for job_id, result, path in results:
        if result and path:
            db.execute("UPDATE jobs SET outreach_path = ? WHERE id = ?", (str(path), job_id))
            count += 1
    db.commit()
    db.close()
    click.echo(f"✅ {count} outreach drafts generated in {drafts_dir}/")
```
